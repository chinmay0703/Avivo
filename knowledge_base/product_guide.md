# Product Guide

## Overview
Our platform is a cloud-based project management solution designed for engineering teams. It helps teams plan sprints, track progress, manage deployments, and collaborate effectively.

## Key Features

### Sprint Planning
- Create and manage sprints with customizable durations (1-4 weeks)
- Drag-and-drop backlog prioritization
- Story point estimation with team velocity tracking
- Automatic sprint reports and burndown charts

### Deployment Tracking
- Integrated with GitHub, GitLab, and Bitbucket
- Visual deployment pipeline with real-time status
- Rollback capability with one-click revert
- Deployment frequency and lead time metrics

### Team Collaboration
- Real-time commenting on tasks and PRs
- @mention notifications across Slack and email
- Shared dashboards customizable per team
- Knowledge base with full-text search

## Pricing Tiers
- **Free**: Up to 5 users, 3 projects, basic features
- **Team**: $12/user/month, unlimited projects, advanced analytics
- **Enterprise**: Custom pricing, SSO, audit logs, dedicated support

## API Access
All plans include REST API access. Rate limits:
- Free: 100 requests/hour
- Team: 1,000 requests/hour
- Enterprise: 10,000 requests/hour

API documentation is available at docs.platform.com/api
